# StockBerry

A modern, clean-architecture Inventory Management System built with **C#** and **WPF**.

## Architecture
StockBerry is built using **N-Tier Architecture** to ensure maintainability and separation of concerns:
* **UI:** WPF (Windows Presentation Foundation) with MVVM pattern.
* **Logic:** Business rules and service layer.
* **Data:** SQLite database integration.
* **Objects:** Core models and data structures.

## Tech Stack
* **.NET 8**
* **WPF (XAML)**
* **SQLite**
* **SHA-256 Authentication**
